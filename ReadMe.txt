FILES INTERNSHIP 2025
The most important files are sorted in 3 different folders:

-GENERATE_DATASETS (MATLAB)
	All MATLAB files and COMSOL files necessary for generating data sets
	for network training. 

-TRAIN_NETWORK (Python)
	Python and COMSOL files necessary for training a neural network
	for predicting the optimal CFL number.

-RUN_NETWORK_RESULTS (Python)
	All Python and COMSOL files necessary for applying and running the
	neural network in COMSOL.
